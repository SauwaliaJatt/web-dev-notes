Node.js Basics

Essential concepts and methods for building server-side applications with Node.js.

require()

Import modules or files into your Node.js application.

Syntax: const module = require('module-name'); // Imports a module

Example:

Importing the built-in 'fs' module to work with the file system.

const fs = require('fs'); // Use fs to read/write files
Node.js Module Types
Module Type Description
Core Modules Built-in modules provided by Node.js (e.g., fs, http).
Local Modules Modules created within your project.
Third-Party Modules Modules installed via npm (e.g., express).
